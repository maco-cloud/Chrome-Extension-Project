# Enable GitHub Pages (one-time, ~2 minutes)

Code is already on `main`. Do this in your browser:

1. Open: **https://github.com/maco-cloud/Chrome-Extension-Project/settings/pages**
2. Under **Build and deployment** → **Source**, choose **GitHub Actions** (not “Deploy from a branch”).
3. Open: **https://github.com/maco-cloud/Chrome-Extension-Project/actions**
4. Click workflow **Deploy website** → **Run workflow** → **Run workflow** (branch `main`).
5. Wait until the run shows a green check (~1–2 min).
6. Back to **Settings → Pages** — you should see:  
   `Your site is live at https://maco-cloud.github.io/Chrome-Extension-Project/`
